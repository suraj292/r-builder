# Tech Stack: Resume Builder

## Frontend
- **React 18+**: Concurrent features for smooth UI updates.
- **TypeScript**: Strict typing for complex schema-driven state.
- **Tailwind CSS**: Utility-first styling for high performance and easy theme injection.
- **Zustand**: Fast, lightweight state management with middleware for persistence and history.
- **dnd-kit**: Flexible drag-and-drop toolkit.

## Utilities
- **Immer**: For immutable state updates in complex objects.
- **Zod**: Runtime schema validation for backend synchronization.
- **Lucide React**: For consistent, lightweight icons.
- **PostCSS**: For advanced CSS transformations.

## Architecture
- **Schema-Driven Rendering**: Decoupled data and UI.
- **Measurement-Based Pagination**: JavaScript engine for precise A4 overflow handling.
- **Registry Pattern**: Centralized block component management.
